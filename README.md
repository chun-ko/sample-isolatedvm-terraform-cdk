# Sample isolated vm Terraform CDK script

## Prerequisite 

* NodeJS
* CDKTF
* Yarn


## Quick Installation

```
npm install --global cdktf-cli@latest
```

## Quick Start for new project
```
mkdir test-project
cd test-project
cdktf init
```

## Deploy this script
```
cdktf deploy
```
